Day 02 – Security Testing and Network Fundamentals

-->Topics Covered

-->Types of Security Testing
- White Box Testing
- Black Box Testing
- Gray Box Testing

--->Security Testing Based on Location
- Internal Testing
- External Testing

-->Computer Networks (By Area)
- LAN – Local Area Network
- MAN – Metropolitan Area Network
- WAN – Wide Area Network

-->Networks Based on Accessibility
- Internal Networks
- External Networks

-->Key Learning
Before performing any security assessment, an ethical hacker must clearly understand the testing type, network scope, and access level. These fundamentals ensure proper planning, legal compliance, and accurate vulnerability identification.
